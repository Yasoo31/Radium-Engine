Detailed Description

template<typename T>
class Ra::Core::Geometry::IndexedGeometry< T >

Simple Mesh structure that handles indexed polygonal mesh with vertex attributes. Each face is indexed with typename T = IndexType. T is assumed to be an Eigen Vector of unsigned ints observable on indices
