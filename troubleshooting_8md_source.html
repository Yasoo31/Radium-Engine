4 # Qt cmake errors
5 In case you run into an error like
6 ~~~
7 By not providing "FindQt5Widgets.cmake" in CMAKE_MODULE_PATH this project
8 has asked CMake to find a package configuration file provided by
9 "Qt5Widgets", but CMake did not find one.
10 ~~~
11 you need to set `CMAKE_PREFIX_PATH`, pointing to the Qt root dir of your commpiler.
12 For example on linux with gcc :
13 ~~~bash
14 $ cmake -DCMAKE_PREFIX_PATH=/opt/Qt/5.x/gcc_64
15 ~~~
16 
17 On windows, using cmake-gui you can use the "add entry" button, adding `CMAKE_PREFIX_PATH`
18 as a string to point to the Qt directory (for example in the default installation :
19 `C:/Qt/5.6/msvc2015_64` )
20 
21 # Crash when starting main application on windows
22 This is usally caused by missing dlls.
23 With Visual Studio, you may need to copy the Qt dlls to Radium bin folder `Bundle-MSVC\bin` or `Bundle-MSVC-Debug\bin`.
